# Agent Bakeoff Artifacts

This directory keeps the committed summary of benchmark outcomes.

Raw per-run logs, session transcripts, copied visible workspaces, and other bulky bakeoff output are intentionally git-ignored. They are useful for local investigation, but they are not part of the durable repository surface.

The canonical committed summary is:

- `final_closed_loop_eval.md`
- `final_closed_loop_eval.json`
